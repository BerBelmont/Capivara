# Capivara Tamagotchi

## Requisitos

- Node.js
- npm
- Expo Go, caso queira testar no celular

Verifique a instalação:

```bash
node --version
npm --version
```

## Instalação

```bash
npm install
```

## Rodar o projeto

```bash
npx expo start
```

Depois escolha uma opção no terminal:

- `w` para abrir no navegador
- `a` para abrir no emulador Android
- ler o QR Code com o Expo Go no celular

## Windows

Se o PowerShell bloquear o `npx`, use:

```bash
npx.cmd expo start
```

## Comandos úteis

Rodar no navegador:

```bash
npx expo start --web
```

Verificar TypeScript:

```bash
npx tsc --noEmit
```
